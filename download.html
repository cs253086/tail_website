One-line install

Or just run this.

Installs the tailos CLI, toolchain helpers, and latest QEMU image.

macOS / Linux
curl --proto '=https' --tlsv1.2 -sSf https://tail-os.com/install.sh | sh
Windows (PowerShell)
iwr -useb https://tail-os.com/install.ps1 | iex
Verify

Check your download.

Every artifact is signed with our release key. Fingerprint: 3F8C 4A7E 9B21 TAIL 0OS0.

shell
sha256sum -c tail-os-v0.1.0.sha256
gpg --verify tail-os-rpi3b-v0.1.0.img.zip.sig tail-os-rpi3b-v0.1.0.img.zip
