Why TAIL OS

Built for the next generation of robotics.

A microkernel designed from first principles — deterministic, safe, and small enough to audit.

Microkernel core

A minimal, auditable kernel with userspace drivers. If it doesn't need to be privileged, it isn't.

Hard real-time

Deterministic EDF scheduler with sub-microsecond IPC jitter. Predictable by design, not by accident.

Memory-safe by construction

Written in safe Rust. No null pointers, no data races, no undefined behavior — enforced by the compiler.

Embedded & portable

Runs on Raspberry Pi 3B today, QEMU virt tomorrow, your custom board next. AArch64 first, more to come.

Safer IPC

Typed message channels with bounded queues. Share memory when you need to, copy when you should.

Predictable latency

No global GC, no background daemons, no surprises. Your control loop runs when it's scheduled — period.

Installation

Up and running in under 5 minutes.

Pick your host OS, grab the toolchain, flash it to a board or boot it in QEMU.

  1. Install prerequisites

    You'll need Rust (nightly), QEMU for AArch64, and the tailos CLI.

    macOS / Linux
    curl --proto '=https' --tlsv1.2 -sSf https://tail-os.com/install.sh | sh
rustup toolchain install nightly
rustup component add rust-src llvm-tools-preview --toolchain nightly
brew install qemu    # or: sudo apt install qemu-system-arm

  2. Fetch the latest image

    shell
    tailos pull --target qemu-aarch64

  3. Boot it

    shell
    tailos run --target qemu-aarch64

    You'll drop into the tailos> shell. Try help.

  1. Download the Pi image

    Grab the latest kernel8.img from the download page.

  2. Copy it to an SD card

    macOS / Linux
    diskutil list                          # find your SD card, e.g. /dev/disk4
diskutil unmountDisk /dev/disk4
sudo dd if=tail-os-rpi3b-v0.1.0.img of=/dev/rdisk4 bs=4m status=progress
sync && diskutil eject /dev/disk4

  3. Boot the Pi

    Insert the card, connect UART (115200 8N1) to GPIO 14/15, power on. You'll see the boot log stream over serial at < 2µs IPC latency.
